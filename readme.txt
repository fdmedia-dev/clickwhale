== Installation ==

This plugin is easy to install and configure.

1. Upload the `clickwhale` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the settings page, which can be found under the menu: ClickWhale

On the settings page, you will find all the instructions and credentials which are necessary to configure the plugin.
